# Playwright Test Agents 技术分享

## 内容

| 文件 | 说明 |
|------|------|
| `index.html` | 网页 PPT，浏览器直接打开即可（科技创新风，紫+青配色） |
| `share-script.md` | 配套逐页分享稿（约 20 分钟） |

## PPT 使用

1. 用浏览器打开 `index.html`
2. 右上角全屏按钮进入全屏模式
3. 鼠标滚轮 / 键盘方向键 / 触摸滑动翻页

### 快捷键

| 按键 | 功能 |
|------|------|
| `↓` / `PageDown` / `空格` | 下一页 |
| `↑` / `PageUp` | 上一页 |
| `Home` | 第一页 |
| `End` | 最后一页 |
| `F`（点击右上角按钮） | 全屏切换 |
